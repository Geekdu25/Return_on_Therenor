Return on Therenor

Return on Therenor est un jeu d'action-aventure en trois dimensions qui est la suite de The legend of Therenor, projet ayant remporté les trophées de la NSI au niveau territorial en 2024. Pour réaliser cette suite, l'équipe a essayé de respecter le thème de l'année qui était Arts & informatique. Bien sûr, même si le projet a necessité d'écrire plusieurs lignes de code, d'autres réalisations artistiques ont été effectuées pour créer ce projet, notamment les modèles 3D.

Scénario du jeu :

Au cours du jeu vous incarnerez un jeune homme vivant sur un village flottant sur pilotis. Très tôt dans le jeu, vous apprendrez que son grand-père a disparu ; vous vous mettrez donc à sa recherche. Mais quelle est a nature de la véritable menace qui pèse sur l'île de Therenor ?
